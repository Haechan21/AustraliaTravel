## Day 6 — 5/29 (금) 블루마운틴 → 그랜드퍼시픽 드라이브 → 시드니

> 6조 기본: 그랜드 캐니언 트랙(S) + 그랜드 퍼시픽 드라이브(S) + 시 클리프 브릿지(A) + 봄보(A) + 키아마(A). ~339km. 렌터카 반납(마감 21:00).
> ☀️ 블루마운틴 — 일출 06:46 / 일몰 17:02 · 시드니 — 일출 06:41 / 일몰 16:59

### 요구사항

- **숙소에서 일출 감상 (06:46)** — 뷰가 좋으니 여유롭게 아침 시간을 보내고 싶다. 농장 풍경 + 아침 서리/안개. 에스프레소 머신으로 커피 내려서 느긋하게.
- **그랜드퍼시픽 드라이브를 석양 드라이브로** — 드라이브 끝(키아마)에서 일몰(~17:00)을 맞이하는 방향.
- 렌터카 반납 마감 21:00 활용 — 기존 16:10 반납에서 늦춰도 OK.

### 변형 일정안

| 시간 | 내용 | 비고 |
|------|------|------|
| 06:46 | 숙소에서 일출 감상 | 농장 뷰 + 아침 서리/안개 |
| 07:00~09:00 | 여유로운 아침 | 에스프레소 커피, 농장 산책, 아침식사(자취) |
| 09:00 | 체크아웃, 출발 | 숙소 → 블랙히스 ~25분 |
| 09:30~12:00 | **그랜드 캐니언 워킹 트랙** (S) | 6km 순환, 2.5시간. 이 여행 최고의 하이킹 |
| 12:15~13:00 | **카툼바에서 점심** | 기존 키아마 점심 → 카툼바로 변경 |
| 13:00 | 카툼바 출발 → 스탠웰 탑스 | 130km, ~1.5시간 (Picton 경유) |
| 14:30~14:50 | **스탠웰 탑스 전망대** (A) | 그랜드퍼시픽 시작점 |
| 15:00 | **그랜드 퍼시픽 드라이브** 시작 (S) | 🌅 여기서부터 석양 드라이브 |
| 15:20~15:50 | **시 클리프 브릿지** (A) | 석양빛 해안 교각 위 도보 |
| 16:10~16:30 | **봄보 헤드랜드** (A) | 주상절리 + 골든아워 파도 |
| 16:40~16:55 | **키아마 블로우홀** (A) | 🌅 일몰(~17:00) 직전 마무리 |
| 17:00 | 키아마 출발 → 시드니공항 | 117km, ~1.5시간 (M1) |
| 18:30 | 시드니공항 주유 + 렌터카 반납 | 21:00 마감 대비 2.5시간 버퍼 |
| 18:50 | Airport Link → 시드니 시내 | ~15분 |

**기존 대비 변경점**:
- 출발 07:00 → 09:00 (+2시간, 숙소 일출·아침 여유)
- 점심: 키아마 → **카툼바**로 이동 (트레킹 직후 자연스러운 식사)
- 그랜드퍼시픽 구간 15:00~17:00 = **골든아워 석양 드라이브**
- 반납: 16:10 → **~18:30** (마감 대비 2.5시간 버퍼)
- 금요일 M1 교통: 17:00 키아마 출발은 남→북 방향이라 정체 영향 제한적 (최대 +30분)
