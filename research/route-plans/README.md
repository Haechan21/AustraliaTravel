# 로드트립 루트 후보 6개 안 — 종합 비교 (v3)

> **작성일**: 2026-03-15 (v3 업데이트)
> **목적**: 시드니공항 출발/반납 (5/24~5/29, 6일) 로드트립 루트 후보를 비교하여 최적 일정 선정
> **평가자**: 효율 전략가 김도현(A) / 감성 탐험가 이서아(B) / 현실주의 비평가 박지우(C)
> **기반**: [TRAVELER_PROFILE.md](../../docs/TRAVELER_PROFILE.md) 여행자 프로필 반영

---

## ⚠️ v2 → v3 주요 변경점

### 중대 사실 오류 정정: 월령(Moon Phase)

v2에서 **전 루트의 은하수 촬영 계획이 잘못된 월령 가정에 기반**하고 있었음이 확인됨.

| 항목 | v2 가정 | 실제 (검증 완료) |
|------|--------|-----------------|
| 신월 | 5/23 (여행 첫날) | **5/16** (여행 1주 전) |
| 보름달 | 미확인 | **5/31** (귀국일 = 보름달!) |
| 여행 기간 달 밝기 | 0~10% (암흑) | **58~99%** (밝음→거의 보름) |
| 은하수 촬영 | 가능 (최적 조건) | **전 기간 불가능** |

**영향**: 4조("스타라이트 루프")의 핵심 전제가 무효화. 전 루트에서 은하수 관련 내용 수정 필요.

> 출처: [moongiant.com](https://www.moongiant.com/moonphases/may/2026/), [phasesmoon.com - May 28](https://phasesmoon.com/moonday28May2026.html)

### 루트 평가 체계화

v2에서는 루트 평가 기준이 명시되지 않았음. v3에서는 은하수 불가 반영 + 프로필 부합도 재검증 + 각 루트 일정 개선을 반영하여 전 루트를 재평가.

## v1 → v2 변경점 (기존 유지)

여행자 프로필 반영으로 전제 조건이 크게 바뀜:
- **첫날 900km 직행 가능** → GC 직행이 더 이상 페널티가 아님
- **매일 350~400km OK** → 중거리 이동일도 관광일로 활용 가능
- **야간 2시간 OK** → 일몰 후 이동에 여유
- ~~**은하수 촬영 희망**~~ → ~~다크스카이 장소 필수 포함~~ **여행 기간 중 은하수 촬영 불가 (§ 월령 정정 참조)**
- **뷰 숙소 투자 의향** → 숙소 자체가 하이라이트 가능
- **좌측통행 경험 있음** → 첫날 해안도로도 가능

## 각 안 한줄 요약

### 4조: 오션 앤 마운틴 그랜드 루프 (83.0점, 1위) — [상세](4조_자유컨셉.md)
**시드니 → GC(직행) → 바이런 → 콥스/Waterfall Way → 포트스테판스 → 블루마운틴(뷰숙소) → 그랜드퍼시픽 → 시드니**
- **S등급 6곳 올클리어** — 유일한 안. 반딧불이 + 블루마운틴 뷰 숙소
- Day 3~4 개선으로 "이동→여유→이동+관광→여유→빡센→중간" 리듬 달성
- 총 ~2,567km. Day 5(540km) 빡센 날이 최대 부담

### 2조: 남쪽해안 그랜드루프 (82.7점, 2위) — [상세](2조_남쪽해안_블루마운틴.md)
**시드니 → 그랜드퍼시픽(S) → 키아마 → 블루마운틴(S, 2일) → 시드니(Vivid Sydney 최대 4일)**
- 최단 이동거리(~630km). 바다→산→도시 3막 서사. 고벳츠 리프 일출
- **Vivid Sydney 최대 4일** — 야간 콘텐츠 최강. 피로·리스크 최저
- S등급 2곳 + 고벳츠 리프 안개 대안(Day 3/4 교체) 탄력 일정

### 3조: 해안+내륙 폭포 루프 (82.3점, 3위) — [상세](3조_북쪽중거리_내륙.md)
**시드니 → GC(직행) → 바이런 → 콥스하버 → Waterfall Way(도리고) → 포트맥쿼리 → 포트스테판스/뉴캐슬 → 시드니**
- Waterfall Way 열대우림이 루트의 하이라이트. 경험 다양성 최고
- S등급 4곳 전원 확정 (뉴캐슬 메모리얼 워크 확정 배치)
- 빡센/여유 교차 완벽 구현. 태킹포인트 골든아워 포토

### 1조: GC 직행 + 남하 해안 (76.7점, 4위) — [상세](1조_GC직행_남하해안.md)
**시드니 → GC(직행) → 바이런 → 콥스/Waterfall Way → 포트맥쿼리 → 포트스테판스 → 시드니**
- 프로필 부합도 높은 안정적 구조. 빡센/여유 교차 자연스러움
- S등급 4곳. 바이런 체류 3시간으로 확대. 커럼빈 선택 추가
- 3조와 유사하나 뉴캐슬 경유 없이 더 단순한 동선

### 5조: 풀 스윙 — 개선 (71.3점, 5위) — [상세](5조_야심작.md)
**시드니 → GC → 바이런 → 콥스/도리고 → 포트스테판스 → 시드니 (블루마운틴/그랜드퍼시픽 삭제)**
- 블루마운틴 삭제로 빡센/여유 교차 달성. S등급 3곳
- 사실상 1조와 유사한 구조가 됨. 독자적 존재 의의 약화
- 1조/3조 대비 차별점이 적어 실질적 선택지에서 후순위

### 6조: 바이런베이 직행 + 블루마운틴 (87.0점) — [상세](6조_바이런직행_블루마운틴.md)
**시드니 → 바이런(직행) → 콥스/Waterfall Way → 포트스테판스 → 블루마운틴(뷰숙소) → 그랜드퍼시픽 → 시드니**
- GC를 과감히 포기하고 바이런 직행으로 이동 부담 대폭 절감
- S등급 6곳 올클리어. 블루마운틴 석양 뷰 숙소 + 그랜드퍼시픽 피날레
- 총 ~2,085km. 일일 평균 ~258km로 프로필 편안한 운전 거리에 부합
