v2에서 **전 루트의 은하수 촬영 계획이 잘못된 월령 가정에 기반**하고 있었음이 확인됨.

| 항목 | v2 가정 | 실제 (검증 완료) |
|------|--------|-----------------|
| 신월 | 5/23 (여행 첫날) | **5/16** (여행 1주 전) |
| 보름달 | 미확인 | **5/31** (귀국일 = 보름달!) |
| 여행 기간 달 밝기 | 0~10% (암흑) | **58~99%** (밝음→거의 보름) |
| 은하수 촬영 | 가능 (최적 조건) | **전 기간 불가능** |

**영향**: 4조("스타라이트 루프")의 핵심 전제가 무효화. 전 루트에서 은하수 관련 내용 수정 필요.

### 루트 평가 체계화

v2에서는 루트 평가 기준이 명시되지 않았음. v3에서는 은하수 불가 반영 + 프로필 부합도 재검증 + 각 루트 일정 개선을 반영하여 전 루트를 재평가.

## v1 → v2 변경점 (기존 유지)

여행자 프로필 반영으로 전제 조건이 크게 바뀜:
- **첫날 900km 직행 가능** → GC 직행이 더 이상 페널티가 아님
- **매일 350~400km OK** → 중거리 이동일도 관광일로 활용 가능
- **야간 2시간 OK** → 일몰 후 이동에 여유
- ~~**은하수 촬영 희망**~~ → ~~다크스카이 장소 필수 포함~~ **여행 기간 중 은하수 촬영 불가 (§ 월령 정정 참조)**
- **뷰 숙소 투자 의향** → 숙소 자체가 하이라이트 가능
- **좌측통행 경험 있음** → 첫날 해안도로도 가능
