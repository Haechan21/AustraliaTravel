# 5월 계절 보정 프레임워크

> 현재 CRITIC.md 기반 3인 평균 점수는 "연중 평균 품질"에 가깝다.
> 이 프레임워크는 2026년 5월 23~30일 방문이라는 조건에서 점수가 달라져야 하는 장소를 식별하고, 보정 방법을 정의한다.

## 1. 5월 하순 NSW 계절 특성 요약

### 1.1 기후

| 요소 | 해안 (시드니~골드코스트) | 내륙 (블루마운틴) |
|------|------------------------|------------------|
| 낮 기온 | 18~25도 (남→북 점진 상승) | 14~17도 |
| 밤 기온 | 9~16도 | 3~6도 (서리 가능) |
| 일몰 | 16:55~17:08 | ~17:00 |
| 월 강수일 | 8~15일 | ~6일 |
| 안개 | 해안 적음 | 빈도 높음 (특히 이른 아침) |

### 1.2 해양

- **수온**: 시드니 20~21도 (서늘), 바이런베이~골드코스트 22~24도 (쾌적)
- **스웰**: 겨울 south-east swell 시작 — 파도 크기/빈도 증가
- **해류**: 겨울 해류 패턴으로 전환, 일부 해변 유속 증가

### 1.3 야생동물

- **혹등고래**: 5월 북상 시작 (6~7월 피크, 5월은 시즌 초반)
- **캥거루**: 5월 활동 증가 (이른 아침/해질녘), 도로 사고 32% 증가
- **코알라**: 연중 관찰 가능 (계절 변동 미미)
- **펭귄**: 겨울 번식기 아님, 관찰 기회 연중 유사
- **반딧불이**: 야외형은 겨울 감소, 터널형(Lithgow)은 연중 가능

### 1.4 식생

- 호주 대부분 상록수 — 가을 단풍 효과 제한적
- 일부 낙엽수(포플러, 단풍나무 등 유럽 도입종)가 있는 정원/공원에서만 단풍
- 5월 하순은 가을 말 — 야생화 시즌 아님 (봄 8~10월이 피크)

### 1.5 관광 환경

- **비수기 진입**: 학교 방학 외 기간, 혼잡도 대폭 감소
- **Vivid Sydney**: 5/22~6/13 — 여행 전 기간 해당! 시드니 도시 야간 관광 가치 극대화
- **운영시간 축소**: 일부 시설 동절기 단축 운영 (17시 마감 등)
- **비수기 가격**: 숙소/투어 할인 가능성

## 3. 보정 적용 방법

### 3.1 산출 공식

```
5월 보정 점수 = 기존 3인 평균 점수 + 계절 보정값
```

- **보정 범위**: -10 ~ +10점 (100점 만점 기준)
- **기존 점수에 직접 가산/감산**하는 방식 채택
- 기준별로 보정하지 않고, 장소 단위로 통합 보정값을 산출하는 이유:
  - 기준별 보정 시 각 페르소나 가중치와 교차하면서 복잡도가 과도하게 증가
  - 계절 요인은 "이 장소가 5월에 더 좋은가/나쁜가"라는 단일 질문에 대한 답이므로 통합 보정이 직관적
  - 단, 보정 근거는 어떤 기준에 영향을 주는지 명시 (추적 가능성 확보)

### 3.2 보정값 산출 구조

각 장소에 대해 다음을 기록한다:

```json
{
  "id": "30097942",
  "name_ko": "키아마 블로우홀",
  "seasonal_adjustment": {
    "total": +7,
    "category": "strong_positive",
    "factors": [
      {
        "criterion": "scenery",
        "delta": "+3",
        "reason": "겨울 스웰로 블로우홀 분출 빈도/높이 극대화, 5~9월이 최적기"
      },
      {
        "criterion": "uniqueness",
        "delta": "+3",
        "reason": "겨울 스웰 블로우홀은 5월에만 가능한 수준의 장관"
      },
      {
        "criterion": "value_for_money",
        "delta": "+1",
        "reason": "비수기로 주차/관람 혼잡 감소"
      }
    ],
    "adjusted_score": 78.5,
    "adjusted_grade": "A"
  }
}
```

### 3.3 보정 후 등급 재산정

보정 후 점수로 전체 110곳을 재정렬하여 퍼센타일 기반 등급을 재부여한다.
기존 등급 기준(S: 상위 5%, A: 5~25%, B: 25~60%, C: 60~90%, D: 하위 10%)을 동일하게 적용한다.

일부 장소는 하나의 요인이 양의 보정을, 다른 요인이 음의 보정을 유발한다. 이 경우 **상쇄 후 순보정값**을 적용한다.

### 5.2 날씨 의존도 표기

계절 보정이 날씨 조건에 크게 좌우되는 장소에는 `weather_dependent: true` 플래그를 추가한다.
이는 "맑은 날에는 보정값대로 좋지만, 비/안개 시에는 보정이 무효화되거나 역전될 수 있음"을 뜻한다.

해당 장소:
- 블루마운틴 전망대 전체 (안개 리스크)
- 해안 절벽 전망대 (비 오면 경관 저하 + 미끄러움)
- 고래 관찰 전망대 (비/안개 시 관찰 불가)

### 5.3 Vivid Sydney 보정의 조건부 적용

Vivid Sydney 보정(+7~10)은 **5/30 시드니 시내 관광일에만 유효**하다.
- 5/24~29 로드트립 기간에는 시드니에 없으므로 Vivid 보정 무의미
- 단, 5/29 저녁 시드니 복귀 후 일부 체험 가능성 있음
- 보정 적용 시 `condition: "5/30 시드니 도시관광일에 방문하는 경우에만 적용"` 명시

### 5.4 고래 관찰 보정의 확률적 성격

5월은 혹등고래 북상 **시작** 시점으로, 6~7월 피크 대비 목격 확률이 낮다.
- 보정 적용 시 "5월 하순 목격 확률 ~40~60% (6~7월 대비)" 명시
- 전망대에서 무료 관찰이므로 못 봐도 손해가 적음 → 보정은 유지하되 과대 적용하지 않음

## 9. 설계 원칙

1. **보정은 후처리**: 기존 3인 평가 체계(CRITIC.md)를 수정하지 않는다. 보정은 최종 점수에 대한 가산/감산이다.
2. **보정 근거 추적 가능**: 모든 보정에 영향 기준(criterion)과 사유(reason)를 명시한다.
3. **과보정 방지**: 보정 범위를 -10~+10으로 제한. 100점 만점에서 10%를 넘는 보정은 원래 평가 자체가 잘못된 것이므로 재평가가 필요.
4. **조건부 보정 명시**: Vivid Sydney처럼 특정 날짜/상황에만 유효한 보정은 condition 필드로 명시.
5. **최종 결정은 사용자**: 보정 후 점수는 의사결정 보조 도구이지, 자동으로 일정을 변경하지 않는다.
