## 1. 호주에서만 먹을 수 있는 음식 체험

### 1-1. 호주 미트파이 — Harry's Cafe de Wheels ⭐

| 항목 | 내용 |
|------|------|
| **위치** | Cowper Wharf Rd, Woolloomooloo (본점) / Haymarket점, Darling Harbour점 등 시내 다수 |
| **가까운 역** | Kings Cross역 도보 10분 (본점) / Town Hall역 (Haymarket점) |
| **대표 메뉴** | **Tiger Pie** — 미트파이 위에 매시드 포테이토 + 머시 피(완두콩) + 그레이비 소스 |
| **가격** | Tiger Pie ~$8.90, 핫도그류 ~$9.90 |
| **토요일 운영** | 본점: 토 9:00am~4:00am / Haymarket점: 토 10:00am~4:00am |
| **추천 이유** | 1945년부터 운영한 시드니 아이콘. 호주 미트파이의 정수를 가장 부담 없이 체험 가능 |
| **동선 연결** | 동선 직접 경유하진 않지만, Haymarket점은 시드니 숙소 체크인 전후 간식으로 활용 가능. 본점은 별도 방문 필요 |
| **출처** | [Harry's 공식](https://www.harryscafedewheels.com.au/), [Yelp Woolloomooloo](https://www.yelp.com/biz/harrys-cafe-de-wheels-woolloomooloo-woolloomooloo) |

> **Tip**: 본점(Woolloomooloo)이 분위기 최고지만 동선에서 벗어남. **Haymarket점**이 Central역/Town Hall역에서 접근성 좋아 Day 6 저녁 전 간식으로 추천.

### 1-2. 피쉬앤칩스 — Doyles on the Beach (왓슨스베이) ⭐

| 항목 | 내용 |
|------|------|
| **위치** | 11 Marine Parade, Watsons Bay |
| **가까운 역** | Watsons Bay 페리 선착장 바로 앞 / 버스 324, 325 |
| **대표 메뉴** | Beer-battered Fish & Chips, 해산물 플래터 |
| **가격** | 테이크아웃 Fish & Chips ~$13.80 / 레스토랑 메인 $30~50 |
| **토요일 운영** | 레스토랑 & 테이크아웃 모두 토요일 영업 |
| **추천 이유** | **1885년 창업**, 시드니에서 가장 오래된 해산물 레스토랑. 하버 뷰(오페라하우스+하버브릿지 조망) + 피쉬앤칩스 조합. 테이크아웃 후 해변 잔디에서 먹으면 최고 |
| **동선 연결** | ✅ **완벽한 동선** — 본다이→더갭 코스털워크 후 왓슨스베이 도착 시 점심으로 최적 |
| **출처** | [Sydney.com 피쉬앤칩스 가이드](https://www.sydney.com/articles/best-fish-and-chips-in-sydney), [Urban List](https://www.theurbanlist.com/sydney/a-list/sydneys-best-fish-and-chips) |

> **대안**: Watsons Bay Hotel도 피쉬앤칩스 + 하버뷰 조합 가능. 좀 더 캐주얼한 펍 분위기.

**본다이 인근 옵션**:
- **North Bondi Fish** (Matt Moran 셰프) — 본다이 해변 조망 + 고급 피쉬앤칩스, 메인 $30~45
- **Fishmongers Bondi** — 캐주얼 테이크아웃, 칼라마리 + 피쉬앤칩스 $15~20

### 1-7. Tim Tam Slam & Vegemite 체험

| 항목 | 내용 |
|------|------|
| **Tim Tam Slam 방법** | 팀탐 양쪽 모서리를 한 입씩 베어 물고 → 따뜻한 커피/핫초코를 빨대처럼 빨아올림 → 녹은 팀탬을 한입에! |
| **구입처** | Woolworths, Coles 등 **슈퍼마켓** 아무 곳이나 (1팩 ~$3.50~5) |
| **Vegemite** | 토스트에 버터를 듬뿍 → Vegemite는 **아주 얇게** 바름. 처음부터 두껍게 바르면 호주인도 역겨워함 |
| **추천 방식** | 슈퍼에서 Tim Tam + 커피 사서 숙소에서 체험. 또는 카페에서 플랫화이트 시키고 팀탬 슬램 직접 시도 |
| **동선 연결** | 숙소 근처 슈퍼에서 구매 → 숙소 체크인 후 또는 아침에 체험 |
| **추천 이유** | 호주인의 소울푸드. 관광 '머스트두'로 꼽히는 체험. 가격 부담 제로 |
| **출처** | [TripAdvisor 포럼](https://www.tripadvisor.com/ShowTopic-g255060-i122-k326172-o10-Tim_Tam-Sydney_New_South_Wales.html) |

> **추가 호주 간식**: Lamington(라밍턴, 초코+코코넛 스폰지케이크), Anzac Biscuit, Caramello Koala도 슈퍼에서 구매 가능.

## 2. 시드니 대표 맛집/식음 명소

### 2-1. 시드니 피시마켓 (New Sydney Fish Market) ⭐⭐

| 항목 | 내용 |
|------|------|
| **위치** | 1 Bridge Road, Glebe (Blackwattle Bay) — **2026년 1월 신축 이전 오픈!** |
| **교통** | 라이트레일 L1 **Wentworth Park역** 하차 / 버스 389, 501 (400m 도보) |
| **토요일 운영** | **7:00am~자정** (금·토) / 개별 매장 영업시간 상이 |
| **가격대** | 피쉬앤칩스 ~$15~20, 오이스터 반다즌 ~$18~25, 해산물 플래터 $30~60 |
| **꼭 먹어볼 것** | |
| | 1. 갓 까낸 **시드니 록 오이스터** + 레몬 |
| | 2. **Get Fish** — 클래식 피쉬앤칩스 |
| | 3. **Tam Jiak** — 말레이시안, 알래스칸 킹크랩 차콰이띠아오 (Junda Khoo 셰프) |
| | 4. **TJ's Po' Boys** — 루이지애나식 프론 롤 |
| | 5. **Cow & the Moon** — 수상 경력 젤라토 (유자 화이트초콜릿 등) |
| **추천 이유** | $836M 투입 신축 마켓, 2026년 1월 오픈 직후라 화제. 세계 3대 피시마켓. 30개+ 매장에서 다양한 해산물 체험 |
| **동선 연결** | ⚠️ 본다이→왓슨스베이→서큘러키 동선에서 **벗어남**. Day 6(금) 저녁 도착 후 또는 Day 7 아침 일찍 별도 방문 필요 |
| **출처** | [TimeOut 신축 가이드](https://www.timeout.com/sydney/news/the-new-sydney-fish-market-where-it-is-how-to-get-there-and-everything-you-need-to-know-011426), [TimeOut 추천 음식](https://www.timeout.com/sydney/news/sydneys-new-836-million-fish-market-is-open-heres-what-to-eat-first-011926) |

> **주의**: 피시마켓은 Glebe에 위치해 본다이~서큘러키 동선과 반대 방향. **Day 7 아침에 먼저 방문(7am~) 후 본다이로 이동**하거나, **Day 6 저녁(금~자정 운영)**에 방문하는 전략 필요.
