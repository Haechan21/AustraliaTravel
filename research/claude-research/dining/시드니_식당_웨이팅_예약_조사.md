## 상세 조사

### 1. Opera Bar (Circular Quay)

**개요**: 오페라하우스 바로 앞, 하버브리지 뷰. Matt Moran 셰프 큐레이션 메뉴. "세계 최고의 비어가든"으로 불림.

| 항목 | 내용 |
|------|------|
| 토요일 영업시간 | 09:00 ~ 01:00 (금토 동일) |
| 예약 가능 여부 | **워크인 전용** — 예약 불가. 도착 시 호스트 데스크 체크인 |
| 온라인 예약 | 없음 (그룹 문의만 book@operabar.com.au) |
| 토요 웨이팅 | 평상시 15~30분, **Vivid 기간 금토 30~60분+** |
| Vivid 영향 | **매우 높음** — 오페라하우스가 Vivid 핵심 거점. 금·토·일이 가장 혼잡 |
| 2인 예상 비용 | $80~120 AUD (약 8.2~12.4만원) — 음료 2잔 + 식사 기준 |

**실전 팁**:
- **오후 4시 이전 도착** 권장. 5시 이후 퇴근 인파 + Vivid 관객이 몰림
- 좌석은 선착순(first-come-first-served). 테이블 3단계 레벨 중 **최상단(물가 가까운 쪽)**이 하버브리지 석양 뷰 최고
- QR 코드 주문 → 카드 자동 결제. **현금 카운터 주문 시 카드 수수료 절약 가능**
- 사람은 계속 들고나므로 끈기 있게 기다리면 자리 남. 2인은 비교적 빨리 앉을 수 있음
- Vivid 기간 대중교통 필수 (주변 도로 통제). 여유 시간 충분히 확보

### 2. Bills Bondi (Bondi Beach)

**개요**: 빌 그랭거의 시그니처 리코타 핫케이크로 유명한 호주 브런치 아이콘. 본다이비치 도보 5분.

| 항목 | 내용 |
|------|------|
| 토요일 영업시간 | 07:00 ~ 22:00 |
| 예약 가능 여부 | **브런치(아침~점심)는 워크인 전용** (8인+ 평일만 예약). 디너(17시~)는 예약 가능 |
| 온라인 예약 | OpenTable (디너만), 전화 (02) 8412 0700 |
| 토요 웨이팅 | **45~90분** (피크 타임 08:30~11:30) |
| Vivid 영향 | 없음 (본다이는 Vivid 권역 외) |
| 2인 예상 비용 | $60~90 AUD (약 6.2~9.3만원) — 브런치 2인 + 커피 |

**실전 팁**:
- **7:00 오픈과 동시에 도착**하면 웨이팅 없이 착석 가능
- 8:30~11:30이 피크. **13시 이후**도 비교적 한산
- 대기 명단에 이름 올리고 본다이비치 산책 가능 (전화 알림)
- 인기 메뉴: 리코타 핫케이크 ($24), 풀 오지 브렉퍼스트 ($28)

### 3. Heritage Hotel (The Rocks)

**개요**: 1914년 개업, 시드니 최장수 호텔 라이센스. 캥거루·에뮤·악어 피자로 유명. 호주 크래프트 맥주 130종+.

| 항목 | 내용 |
|------|------|
| 토요일 영업시간 | 11:30 ~ 22:00 |
| 예약 가능 여부 | **불필요** — 펍 스타일, 워크인. 바 좌석 + 테이블 |
| 토요 웨이팅 | **낮음** — 펍이라 회전율 높음. 피크 시 10~15분 대기 가능 |
| Vivid 영향 | 간접 (The Rocks 전체 유동인구 증가하나, 관광 펍이라 큰 차이 없음) |
| 2인 예상 비용 | $60~80 AUD (약 6.2~8.2만원) — 피자 2판 + 맥주 |

**주요 메뉴 가격**:
| 피자 | Small | Large |
|------|-------|-------|
| Coat of Arms (에뮤+캥거루) | $33 | $38 |
| Pepper Kangaroo | $27 | $32 |
| Crocodile | $29 | $34 |
| Margherita | $20 | $25 |

**실전 팁**:
- 토요일 저녁 **라이브 뮤직** 있음 → 분위기 좋음
- 21~23시 **Late Night Cocktail Hour** ($15 칵테일)
- Coat of Arms 피자가 시그니처 — 하프 에뮤 + 하프 캥거루로 두 가지 맛 한번에
- The Rocks 마켓(토요일 10~17시)과 연계 방문 추천

### 4. The Glenmore (The Rocks)

**개요**: 1921년 건축, The Rocks 아이코닉 루프탑 바. 오페라하우스·하버브리지 180도 파노라마 뷰.

| 항목 | 내용 |
|------|------|
| 토요일 영업시간 | 11:00 ~ 01:00 |
| 예약 가능 여부 | **워크인 전용** (루프탑). 이벤트 문의는 가능 |
| 토요 웨이팅 | **루프탑 30분+** (저녁 피크 시). 1층 바는 비교적 여유 |
| Vivid 영향 | **매우 높음** — 루프탑에서 Vivid 조명 직접 감상 가능. 특별 Vivid 장식 |
| 2인 예상 비용 | $50~80 AUD (약 5.2~8.2만원) — 음료 + 가벼운 식사 |

**실전 팁**:
- **17시 이전 도착**하여 루프탑 자리 선점 필수. 18시부터 DJ 시작되며 급속히 혼잡
- 루프탑은 면적이 좁아 **1-in-1-out 운영 가능성** 있음 (특히 Vivid 기간)
- 개폐식 지붕 있어 우천 시에도 이용 가능
- 라운지 층(중간층)에서도 하버 뷰 가능 — 루프탑 만석 시 대안
- 리뷰에서 "극도로 시끄럽고 붐빈다"는 의견 다수 → 조용한 대화보다는 분위기 즐기기용
- Vivid 기간에는 각 층에 특별 장식 추가 → 루프탑 아니어도 즐길 요소 있음

### 5. Doyles on the Wharf (Watsons Bay)

**개요**: 1885년 개업, 시드니 최고령 해산물 레스토랑 가문. 워프 테이크아웃과 레스토랑 별도 운영.

| 항목 | 내용 |
|------|------|
| 토요일 영업시간 | **테이크아웃**: 10:45 ~ 17:00 / **레스토랑(On the Beach)**: 11:45 ~ 16:00 (토·일) |
| 예약 가능 여부 | 테이크아웃: 불필요 / 레스토랑: **예약 강력 권장** |
| 온라인 예약 | 레스토랑: doyles.com.au |
| 토요 웨이팅 | 테이크아웃: **10~20분** (주문 후 조리 대기) / 레스토랑: 예약 없으면 30분+ |
| Vivid 영향 | 없음 (왓슨스베이는 Vivid 권역 외) |
| 2인 예상 비용 | 테이크아웃: $40~50 AUD (약 4.1~5.2만원) / 레스토랑: $120~160 AUD (약 12.4~16.5만원) |

**테이크아웃 주요 가격**:
- Fish & Chips: $19.90
- Barramundi: $18.80
- Flathead: $19.80

**실전 팁**:
- 테이크아웃은 **카운터 주문 → 번호표 대기** 방식. 조리 시간 10~15분
- "미리 만들어둔 메뉴(King Prawn Salad, Prawn Roll)"를 주문하면 대기 시간 단축
- 페리로 서큘러키에서 왓슨스베이까지 약 25분 → 페리 자체가 하버 크루즈
- 테이크아웃 후 **로버트슨 파크 잔디밭**에서 피크닉 추천 (하버 뷰)
- 레스토랑은 비싸지만 테이크아웃 맛은 동일

**개요**: 오페라하우스 Western Broadwalk. 원주민 요리 전문 셰프 Mark Olive의 부시 터커 레스토랑. 2025 NSW Tourism Restaurant 수상.

| 항목 | 내용 |
|------|------|
| 토요일 영업시간 | 런치 11:30~14:30 / 디너 17:00~20:30 / 하이티 토·일 오전+오후 |
| 예약 가능 여부 | **예약 강력 권장** |
| 온라인 예약 | middensydney.com.au/reservation 또는 전화 02 9250 7220 |
| 토요 웨이팅 | 예약 시 없음. 워크인 시 20~40분 (Vivid 기간 더 길어질 수 있음) |
| Vivid 영향 | **직접** — 오페라하우스 내 위치. Vivid 관객 유입으로 매우 혼잡 예상 |
| 2인 예상 비용 | 아라카르트: $150~250 AUD (약 15.5~25.8만원) / 데구스타시옹: $440+ AUD |

**주요 메뉴 가격**:
- Wattle Seed Rubbed Kangaroo Fillet: $42
- Crispy Skin Barramundi: $45
- Seared Duck Breast: $46
- Pepper Berry Kangaroo Rack (2인): $105
- 5코스 데구스타시옹: $220/인

**실전 팁**:
- **지금 바로 예약** — Vivid 기간 토요일은 매진 가능성 높음
- Tour & Dine 패키지(오페라하우스 투어 + 식사) 가능
- Native High Tea(토·일)도 독특한 경험 — 부시 터커 재료 활용
- Vivid 기간 식사 후 바로 Vivid 라이트 워크 가능 (위치 최적)

### 7. The Grounds of Alexandria

**개요**: "세계에서 가장 인스타그래머블한 카페". 농장 테마, 3개 다이닝 공간, 커피 로스터리.

| 항목 | 내용 |
|------|------|
| 토요일 영업시간 | 07:30 ~ 16:00 |
| 예약 가능 여부 | **온라인 예약 가능 (4인 이상)**. 2인은 워크인 |
| 온라인 예약 | thegrounds.com.au (카드 등록 필수, 2시간 전 취소 미이행 시 $30/인 차지) |
| 토요 웨이팅 | **60분+** (피크 09~12시) |
| Vivid 영향 | 없음 (알렉산드리아는 Vivid 권역 외) |
| 2인 예상 비용 | $60~100 AUD (약 6.2~10.3만원) |

**브런치 세트 메뉴**:
- Sweet Menu: $30/인 (핫케이크, 치아 푸딩, 패스트리 + 커피)
- Sweet & Savory Menu: $49/인 (위 + 연어, 프리터, 베이컨에그)

**실전 팁**:
- **7:30 오픈과 동시 도착**이 유일한 웨이팅 회피법
- 전자 대기 리스트 → 문자 알림 → 기다리는 동안 농장·정원·마켓 구경 가능
- 키오스크에서 테이크아웃 커피+음식 가능 (줄 없음)
- **주말 브런치 예약은 4인 이상만 가능** → 2인 여행자는 워크인 필수
- 시드니 중심부에서 차로 15분, 대중교통은 Green Square역에서 버스

| 항목 | 내용 |

| 토요 웨이팅 | **혼잡** — 개장 직후라 주말 인파 극심. 인기 가게 10~20분 대기 |
| Vivid 영향 | 간접 (Blackwattle Bay, Vivid 주요 동선과 약간 떨어짐) |
| 2인 예상 비용 | $50~80 AUD (약 5.2~8.2만원) — 해산물 2인분 + 음료 |

**실전 팁**:

- 개장 첫해라 주말 방문객 폭주 중 (개장 1주 23만명 방문)
- 교통: Light Rail L1 **Wentworth Park역** 하차 후 도보 400m. 주차 극히 제한

- 2층 테라스에서 Blackwattle Bay 뷰 + 식사 가능
- 신선 해산물 구매 + 즉석 조리 서비스도 있음

- **평일이 훨씬 여유**. 주말은 인파 때문에 이동도 힘듦
- 2인이면 각자 다른 스톨에 줄서서 시간 절약
- **Burn City Smokers** (브리스킷), **Hoy Pinoy** (필리핀 BBQ)가 인기 스톨 → 줄이 김
- 오픈파이어 쿠킹 데모는 별도 공간 → 유명 셰프(Mark Best, Ben Devlin 등) 시그니처 요리 구매 가능
- 대중교통 필수. Central역 또는 Wynyard역에서 도보 접근

### 10. love.fish (바랑가루)

| 예약 가능 여부 | **예약 권장** |
| 온라인 예약 | bookings@lovefish.com.au 또는 전화 (02) 8077 3700 |
| 토요 웨이팅 | 예약 없으면 **20~40분** (금·토 저녁 피크) |
| Vivid 영향 | **직접** — 바랑가루가 Vivid 핵심 구역. Fire Kitchen과 인접 |
| 2인 예상 비용 | $80~140 AUD (약 8.2~14.4만원) / Feed Me 셰어링: $69/인 ($142 총) |

- Feed Me 셰어링 메뉴 ($69/인)가 가성비 좋고 다양한 해산물 맛볼 수 있음

- 리뷰 평판 좋음 (Food·Service·Atmosphere 모두 높은 평점)

### 11. Harry's Cafe de Wheels (Haymarket점)

**개요**: 1938년 개업, 시드니 아이콘 미트 파이 가판대. 프랭크 시나트라, 엘튼 존, 해리 왕자도 방문.

| 항목 | 내용 |
|------|------|
| 토요일 영업시간 | 10:00 ~ 04:00 (심야 영업) |
| 예약 가능 여부 | **불필요** — 테이크아웃 가판대 |
| 토요 웨이팅 | **5~10분** (주문~수령). 식사 시간대 약간 더 길 수 있음 |
| Vivid 영향 | 간접 (Haymarket은 Vivid 동선과 약간 떨어짐) |
| 2인 예상 비용 | $20~30 AUD (약 2.1~3.1만원) |

**주요 가격**:
- Tiger Pie (시그니처: 파이 + 매시 + 머시 피 + 그레이비): $11.50
- Beef Mince Pie: $7.50
- Pie Deal (파이 + 음료): $15~18.50

**실전 팁**:
- Woolloomooloo 본점이 유명하지만 Haymarket점이 접근성 좋음
- Tiger Pie가 시그니처 → 반드시 주문
- 늦은 밤 술 마신 후 해장용으로도 인기
- 테이크아웃 전용이라 근처 공원/벤치에서 먹어야 함

### 12. Icebergs Dining Room (Bondi Beach)

**개요**: 본다이 해변 절벽 위, 수영장 뷰의 파인 다이닝 이탈리안. World's 50 Best Discovery 선정.

| Vivid 영향 | 없음 (본다이는 Vivid 권역 외) |

**실전 팁**:

- 토요일 석양 시간대(17~18시)도 인기 → 런치 or 이른 디너 권장

### 13. Whalebridge (Circular Quay)

**개요**: 서큘러키 프롬나드 프렌치 레스토랑. 오페라하우스·하버브리지 직접 조망.

| 항목 | 내용 |
|------|------|
| 토요일 영업시간 | 12:00 ~ late (매일, 공휴일 포함) |
| 예약 가능 여부 | **예약 권장, 워크인도 환영** |
| 온라인 예약 | whalebridge.com.au/reservations 또는 SevenRooms |
| 토요 웨이팅 | 예약 시 없음. 워크인 시 **10~20분** |
| Vivid 영향 | **직접** — CQ 프롬나드 위치, Vivid 라이트 워크 바로 옆 |
| 2인 예상 비용 | $120~180 AUD (약 12.4~18.5만원) / 프리시어터 메뉴: $59/인 |

**주요 메뉴**:
- 프리시어터 2코스: $59/인
- French Onion Soup + Gruyere Crouton
- Steak Frites (가성비 좋다는 리뷰 다수)
- Lobster with Mustard & Gruyere
- Mussels Cream, White Wine, Garlic

**실전 팁**:
- **프리시어터 메뉴 ($59/인)**가 가성비 최고 → 이른 저녁(17~18시) 추천
- 공휴일 15% 서차지, 카드 1.5% 서차지 적용
- 서비스 속도가 바쁠 때 느려진다는 리뷰 있음 → 시간 여유 두기
- Vivid 라이트 워크 시작 전 식사 → 워크 감상 동선이 이상적
