# 포트맥쿼리 식당 리서치

> 조사일: 2026-03-18
> 대상: Day 3 저녁 + Day 4 아침 (5월 하순)
> 상황: Day 3 오후 도착 (Dorrigo/Waterfall Way에서), Tacking Point Lighthouse 일몰 후 저녁 식사. Day 4 아침 간단히 먹고 Port Stephens으로 출발 (~250km)

---

## 1. 저녁 식사 후보 (Day 3 Dinner)

### 1-1. The Stunned Mullet ⭐ 최우선 추천

| 항목 | 내용 |
|------|------|
| **유형** | 파인다이닝 / 모던 오스트레일리안 시푸드 |
| **위치** | 24 William St, Town Beach 바로 앞 (Tacking Point에서 차 10분) |
| **가격** | 1인 $110~160 (3코스 세트 메뉴) |
| **Google 평점** | 4.7/5 (Restaurant Guru 3,503건 리뷰) |
| **영업시간** | 화~토 저녁 (일·월 휴무 — 5월 방문 요일 확인 필수) |
| **예약** | 필수 (인기 레스토랑) |

**시그니처 메뉴:**
- 전용 굴 메뉴 (dedicated oyster menu)
- Seared scallops with corn salsa
- Crispy king prawns
- Glacier 51 toothfish (dashi-style broth)
- Morton Bay Bug fettuccine
- Milly Hill lamb
- Black Forest cake with cherry sorbet

**분위기:** Town Beach가 내려다보이는 밝은 다이닝룸. Point Plomer까지 해안 조망. 2005년 설립, Good Food Guide Chef's Hat 보유 (2013~), Gourmet Traveller 와인 리스트 3 Glasses (2018~). TripAdvisor Best of the Best (2020~).

**와인:** 메뉴보다 와인 리스트가 방대함. 로컬~글로벌 와인·맥주 셀렉션.

**부정 리뷰 패턴:**
- "Stunned Wallet"이라는 별명 — 가격이 높다는 인식
- 간헐적 서비스 딜레이 보고
- 전반적으로 부정 리뷰 극소수

**커플 적합도:** 최상. 일몰 후 해안가 파인다이닝. 로맨틱한 분위기 + 수상 경력 요리. 특별한 저녁 식사에 적합.

### 1-2. Whalebone Wharf

| 항목 | 내용 |
|------|------|
| **유형** | 시푸드 전문 / 워터프론트 |
| **위치** | 269 Hastings River Dr (Hastings River 수면 위에 위치) |
| **가격** | 메인 ~$49, 1인 $50~80 예상 |
| **Google 평점** | 4.5/5 (Restaurant Guru 2,363건) |
| **영업시간** | 매일 12:00~22:30, 디너 17:15~ |
| **예약** | 권장 |

**시그니처 메뉴:**
- Seafood chowder
- Barramundi
- Fruits de mer (해산물 모둠)
- Lobster
- 해산물 플래터

**분위기:** 40년 이상 된 아이코닉 레스토랑. Hastings River 위에 자리잡아 수면 위 식사 느낌. 탁 트인 수면 조망. 일몰 시간대 특히 분위기 좋음.

**부정 리뷰 패턴:**
- 가격 대비 양이 적다는 의견 일부
- 관광지 레스토랑 특유의 가격 프리미엄

**커플 적합도:** 상. 리버뷰 + 시푸드 = 로맨틱. 단, Stunned Mullet보다 격식 덜함.

### 1-3. Bills Fishhouse + Bar

| 항목 | 내용 |
|------|------|
| **유형** | 캐주얼 시푸드 / 바 |
| **위치** | 2/18-20 Clarence St, CBD 중심 |
| **가격** | 1인 $30~50 (굴 개당 $4, 메인 ~$40) |
| **Google 평점** | 4.6/5 |
| **영업시간** | 확인 필요 (저녁 영업) |
| **예약** | BYO 가능 |

**시그니처 메뉴:**
- Charred octopus
- Yellowfin tuna tacos
- Whole baked local snapper (~$40)
- Oysters ($4 each)
- Char-grilled seafood 전반

**분위기:** 캐주얼하면서 세련된 도시형 분위기. 따뜻한 조명 + 녹색 식물 인테리어. 로컬 식재료 중심 "sea-to-plate" 컨셉. BYO 가능해서 와인을 가져가면 비용 절감.

**부정 리뷰 패턴:**
- 일부 "기대에 못 미침" 리뷰 존재
- 전반적으로 호평 우세

**커플 적합도:** 상. Stunned Mullet보다 캐주얼하고 저렴하지만 품질 좋음. BYO로 와인 가져가면 가성비 우수.

## 3. 특별 식문화 체험

### 3-1. The Big Oyster (Shuck Shack)

| 항목 | 내용 |
|------|------|
| **유형** | 해산물 직판장 + 카페 |
| **위치** | 315 Hastings River Dr, Fernbank Creek (시내에서 차 10분) |
| **영업시간** | 매일 9:00~17:00 (핫푸드: 금~일 12:00~14:00만) |
| **가격** | 굴 1다스 ~$15~20, 새우 버킷 등 |

**특징:** Hastings River 위에 떠 있는 파란 보트 셰드. 나무 보행교로 접근. 양식장에서 직접 수확한 Sydney Rock oysters 판매. 옆 폰툰에서 해산물 플래터 먹을 수 있음. 포트맥쿼리 아이콘.

**추천 시나리오:** Day 3 도착 직후 (오후) 가벼운 간식으로 방문. 굴 한 다스 + 새우 버킷 → Tacking Point 일몰 전 간식. 단, 핫푸드는 금~일만이라 요일 확인 필요.

### 3-2. Rockin' Oysters — 굴 양식장 투어

- Hastings River 상류 Dunbogan 근처 양식장 투어
- James Wood 운영, 강 위에서 Sydney Rock oysters 직접 수확 체험
- 투어 후 James의 셰드에서 해산물 피스트 (굴 다양한 조리법 + 로브스터 + 새우 + 아기 문어)
- 사전 예약 필수, 시간 여유 필요 (Day 3 오후 일정과 충돌 가능성)

## 5. 5월 특이사항

- **겨울 영업시간:** 일부 레스토랑 영업일/시간 축소 가능. 특히 Stunned Mullet (일·월 휴무)
- **기온:** 5월 하순 저녁 15~18도 → 야외석보다 실내석 선호할 수 있음
- **비수기 장점:** 예약 쉬움, 대기 짧음, 로컬 분위기
- **일몰 시간:** 5월 하순 ~17:10 → Tacking Point 일몰 후 17:30~18:00에 저녁 식사 시작 가능
