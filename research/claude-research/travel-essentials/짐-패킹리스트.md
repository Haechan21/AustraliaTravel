
# 호주 여행 짐싸기 & 패킹 체크리스트

> **대상**: 한국인 2명 / 2026년 5월 23~31일 (9일) / 시드니 + NSW 해안 로드트립
> **항공**: 대한항공 KE401 이코노미
> **기후**: 호주 가을 10~22°C, 블루마운틴 최저 5°C, 일몰 17:00
> **액티비티**: 하이킹, 해변, 서핑 체험, 고래관찰, 야시장
> **조사일**: 2026-03-19 | **환율**: 1 AUD ≈ 1,030원 (SSOT)

---

## 1. 항공 수하물 규정 (대한항공 이코노미)

### 위탁수하물 (Checked Baggage)
| 항목 | 규정 |
|------|------|
| 개수 | **2개** (호주 노선 = 미주 외 국제선) |
| 무게 | 각 **23kg** 이하 |
| 크기 | 3변의 합 **158cm(62in)** 이내 (손잡이·바퀴 포함) |
| 초과 요금 | 개당 약 ₩60,000~200,000 (구간·초과 정도에 따라 상이) |

### 기내수하물 (Carry-on Baggage)
| 항목 | 규정 |
|------|------|
| 개수 | **1개** (휴대가방) + **1개** (개인소지품) |
| 무게 | 합산 **12kg** 이내 (KE401 기준, 일반적으로 10kg이나 노선에 따라 12kg) |
| 휴대가방 크기 | 55 x 40 x 20cm / 3변의 합 115cm 이내 |
| 개인소지품 크기 | 40 x 30 x 15cm (앞좌석 아래 수납) |

### 기내 액체류 제한
- 개별 용기 **100ml 이하**, 총 **1L 이하**
- **투명 지퍼백 1개**에 담아 보안검색 시 별도 제출
- 용기 표시 용량 기준 (잔여량 무관 -- 200ml 용기에 50ml만 남아도 반입 불가)
- **예외**: 처방약, 유아식은 100ml 초과 가능 (보안검색 시 신고)

### 보조배터리 규정 (2025년 3월 강화)
| 용량 | 반입 |
|------|------|
| **100Wh 이하** | 기내 반입 가능 (최대 5개) |
| **100~160Wh** | 항공사 사전 승인 필요 (최대 2개) |
| **160Wh 초과** | **반입 금지** |
| 위탁수하물 | **전면 금지** (반드시 기내 휴대) |

> **2025년 3월 신규 규정**: 보조배터리는 좌석 위 선반(오버헤드빈) 보관 금지. 승객이 직접 소지하고, 단자는 절연테이프로 감거나 보호 파우치/지퍼백에 넣어 보관. 기내 충전 행위 금지.

## 2. 호주 반입 금지 & 신고 품목 (한국인 필독)

> 호주는 세계에서 가장 엄격한 검역 규정을 운영한다. **미신고 적발 시 현장 벌금 AUD 660~6,600 (심각도별 4단계) + 비자 취소 가능**.

### 반입 절대 금지
- ☐ 생과일, 생야채
- ☐ 생육류, 가금류, 돼지고기 (가공 포함)
- ☐ 달걀 및 달걀 제품
- ☐ 씨앗, 살아있는 식물, 흙이 묻은 물품
- ☐ 육포, 육송, 육송이 든 과자/에그롤
- ☐ 고기 건더기가 든 라면 (너구리, 짜파게티 건더기스프 등)
- ☐ 고기가 든 만두, 즉석 수프
- ☐ 꿀, 벌집 제품

### 신고 후 반입 가능 (Declare 라인)
- ☐ 순면 라면 (건더기 없는 것: 진라면 순한맛 등) -- **반드시 신고**
- ☐ 햇반 (밀봉 상태) -- **반드시 신고**
- ☐ 공장 밀봉 과자, 사탕, 초콜릿 -- **반드시 신고**
- ☐ 티백, 분말 커피 -- **반드시 신고**
- ☐ 개인 상비약 (처방전 영문 지참 권장)

### 한국인이 자주 걸리는 사례
1. **폐백 음식**: 육포, 밤, 대추 -- 결혼 후 신혼여행 시 다수 적발
2. **라면 건더기**: 쇠고기/돼지고기 성분 포함 여부 꼭 확인
3. **김치, 젓갈**: 발효 식품은 검역 대상
4. **한약**: 동물성 성분 포함 시 반입 금지
5. **나무 공예품**: 흙/껍질이 붙어있으면 압수

### 입국 팁
- 입국카드(Incoming Passenger Card)에 **음식물 소지 여부 정직하게 체크**
- 불확실하면 **무조건 신고** (신고 후 통과는 벌금 없음, 미신고 적발은 벌금)
- 신고 품목이 있으면 **Red Channel(Declare 라인)** 으로 이동

## 3. 의류 -- 5월 호주 레이어드 전략

> 호주 5월 = 한국 11월 초 체감. 낮에는 쾌적하지만 아침·저녁·해안은 춥다.

### 기본 레이어드 구성
| 레이어 | 역할 | 추천 아이템 |
|--------|------|-------------|
| 베이스 | 흡습·속건 | 기능성 긴팔 티셔츠, 얇은 메리노울 |
| 미드 | 보온 | 플리스 집업, 얇은 니트, 후드집업 |
| 아우터 | 방풍·방수 | 고어텍스 방수자켓 (트레킹 겸용) |
| 추가(한랭지) | 보온 강화 | 경량 패딩 (블루마운틴 필수) |

### 의류 체크리스트

**상의**
- ☐ 기능성 긴팔 티셔츠 x 3~4
- ☐ 반팔 티셔츠 x 2 (낮 활동용)
- ☐ 얇은 니트/플리스 x 2
- ☐ 후드집업 또는 가디건 x 1
- ☐ 경량 패딩 (컴팩트 접이식) x 1
- ☐ 방수 윈드브레이커/고어텍스 자켓 x 1

**하의**
- ☐ 긴바지 (청바지 or 치노) x 2
- ☐ 트레킹 팬츠 (속건·신축성) x 1
- ☐ 반바지 x 1 (해변·서핑용)
- ☐ 레깅스 또는 기능성 바지 x 1 (하이킹 레이어)

**속옷·양말**
- ☐ 속옷 x 5~6 (중간 세탁 고려)
- ☐ 기능성 양말 x 4~5 (등산 전용 양말 포함)
- ☐ 얇은 히트텍/내의 x 1 (블루마운틴용)

**수영·해양**
- ☐ 수영복 x 1
- ☐ 래쉬가드 상의 x 1 (서핑 체험, 자외선 차단)
- ☐ 비치타월 or 속건 타월 x 1

**신발**
- ☐ 트레킹화 (방수, 발목 지지) x 1 -- **가장 중요한 아이템**
- ☐ 운동화/스니커즈 x 1 (시내·일상용)
- ☐ 슬리퍼/샌들 x 1 (숙소·해변용)

**모자·액세서리**
- ☐ 챙넓은 모자 또는 캡 x 1 (자외선 차단)
- ☐ 비니 x 1 (블루마운틴 한랭 대비)
- ☐ 선글라스 (UV400) x 1
- ☐ 머플러/넥워머 x 1 (선택)
- ☐ 장갑 (얇은 것) x 1 (블루마운틴 새벽용, 선택)

## 4. 전자기기

### 충전·전원
- ☐ **호주 전원 어댑터 (Type I)** x 2 -- 호주는 3핀 사선형 플러그, 240V/50Hz
- ☐ 멀티탭 (한국 플러그 3~4구) x 1 -- 어댑터 1개로 기기 여러 개 충전
- ☐ USB 멀티 충전기 (USB-A + USB-C) x 1
- ☐ 보조배터리 (10,000~20,000mAh, 100Wh 이하) x 1~2
- ☐ 충전 케이블 (USB-C, Lightning 등) x 2~3
- ☐ 차량용 시거잭 충전기 (USB-C + USB-A) x 1

### 기기
- ☐ 스마트폰 x 2
- ☐ 카메라 + 여분 배터리 + 메모리카드 (선택)
- ☐ 노트북/태블릿 (선택)
- ☐ 이어폰/헤드폰 x 2

### 통신
- ☐ 호주 현지 유심 또는 eSIM (Optus/Telstra/Vodafone -- 사전 구매 권장)
- ☐ 포켓 와이파이 (선택, 2인이면 eSIM 각자가 편리)

> **Tip**: 호주 전압(240V)은 한국(220V)과 유사해 변압기 불필요. 어댑터(플러그 모양 변환)만 있으면 됨.

## 5. 세면도구 & 의약품

### 세면도구 (기내 액체류 100ml 제한 주의)
- ☐ 여행용 샴푸/린스 (100ml 이하 or 고체 샴푸바)
- ☐ 바디워시 (100ml 이하)
- ☐ 세안제/폼클렌징 (100ml 이하)
- ☐ 치약 (100ml 이하) + 칫솔 x 2
- ☐ 선크림 SPF 50+ (호주 자외선 강력 -- 현지 구매도 추천)
- ☐ 보습 로션 (100ml 이하)
- ☐ 립밤 (건조 대비)
- ☐ 면도기 + 면도크림
- ☐ 데오드란트
- ☐ 투명 지퍼백 1L (기내 액체류 보관용) x 2

> **Tip**: 선크림, 샴푸 등은 호주 현지 약국(Chemist Warehouse)에서 저렴하게 구매 가능. 짐 줄이기 전략으로 현지 구매도 고려.

### 상비약
- ☐ 종합감기약 (타이레놀/판콜 등)
- ☐ 해열진통제 (이부프로펜/아세트아미노펜)
- ☐ 소화제/위장약 (훼스탈, 겔포스 등)
- ☐ 지사제 (정로환, 스멕타)
- ☐ 멀미약 (렌터카 동승자용)
- ☐ 밴드/반창고 (트레킹 물집 대비)
- ☐ 소독약/항생 연고 (포비돈, 후시딘)
- ☐ 근육통 파스 (트레킹 후)
- ☐ 알레르기약 (항히스타민제)
- ☐ 모기 기피제 (해안·숲 지역)
- ☐ 비상용 처방약 (개인 복용 약물 -- 영문 처방전 지참)

> **호주 반입 주의**: 개인 의약품은 반입 가능하나, 코데인 함유 진통제(일부 타이레놀 등)는 처방전 필요. 한약(동물성 성분 포함 시 금지). 상비약은 원래 포장 그대로 지참 권장.

## 6. 야외활동 장비

### 하이킹 (블루마운틴, 해안 트레일)
- ☐ 트레킹화 (방수, 발목 보호 -- 위 신발 항목과 동일)
- ☐ 트레킹 폴 (접이식, 선택 -- 위탁수하물로)
- ☐ 소형 배낭 (20~30L, 데이팩)
- ☐ 물통/텀블러 x 2
- ☐ 헤드랜턴 (일몰 17:00으로 빠름, 블루마운틴 필수)
- ☐ 방수 배낭 커버 (우천 대비)
- ☐ 간식 (에너지바, 견과류 -- 현지 구매)

### 해변·서핑
- ☐ 수영복 + 래쉬가드 (위 의류 항목 참조)
- ☐ 방수팩/드라이백 (스마트폰 보호)
- ☐ 아쿠아슈즈 (선택, 암반 해변용)
- ☐ 서핑 체험은 장비 대여 포함이 일반적 (별도 준비 불필요)

### 고래관찰
- ☐ 방풍 자켓 (바다 위 바람 강함)
- ☐ 멀미약 (선박 탑승 전 30분 복용)
- ☐ 쌍안경 (선택)
- ☐ 카메라 줌렌즈 (선택)

### 공통
- ☐ 선글라스 UV400
- ☐ 선크림 SPF 50+ (2시간마다 재도포)
- ☐ 챙넓은 모자
- ☐ 방수자켓 (고어텍스 권장)

## 7. 차량용 준비물 (렌터카 6일)

### 필수
- ☐ **국제운전면허증 (IDP)** + 한국 운전면허증 원본
- ☐ 스마트폰 거치대 (흡착식 or 에어벤트형) x 1
- ☐ 차량용 USB 충전기 (시거잭) x 1
- ☐ USB 케이블 (길이 1m 이상) x 2
- ☐ Google Maps 오프라인 지도 (사전 다운로드)

### 편의
- ☐ 쿨러백/아이스박스 (접이식, 음료·간식 보관)
- ☐ 담요 x 1 (고래관찰·해변 피크닉·차내 낮잠)
- ☐ 쓰레기봉투 (차량 내 정리)
- ☐ 물티슈/키친타월
- ☐ 차량용 방향제 (선택)
- ☐ 우산 x 2 (차에 상비)

### 안전
- ☐ 비상 삼각대 (렌터카에 포함 여부 확인)
- ☐ 렌터카 계약서/보험 서류 사본
- ☐ 긴급 연락처 메모 (렌터카 회사, 보험, 한국 대사관)

> **Tip**: 호주는 좌측통행·우핸들. 로터리(라운드어바웃)가 많으니 사전 연습 필수. 시골 도로에서 캥거루 충돌 주의 (새벽·해질녘).

## 8. 편의용품

### 기내용
- ☐ 목베개 (메모리폼 or 공기주입식)
- ☐ 안대 x 2
- ☐ 귀마개 or 노이즈캔슬링 이어폰
- ☐ 기내용 슬리퍼 (압축 양말도 가능)
- ☐ 기내용 보습 미스트 (100ml 이하)
- ☐ 볼펜 (입국카드 작성용)

### 세탁 (9일 장기여행)
- ☐ 세탁세제 (소분 or 시트형 세제)
- ☐ 세탁망 x 1
- ☐ 빨래줄 + 집게 (선택)
- ☐ 드라이어 시트 (선택)

> **Tip**: 호주 숙소(Airbnb, 모텔) 대부분 세탁기·건조기 구비. 코인런드리도 쉽게 찾을 수 있어 옷은 4~5일분만 가져가고 중간 세탁 권장.

### 기타 편의
- ☐ 여행용 압축팩 (짐 부피 축소)
- ☐ 지퍼백 (다용도, 여러 사이즈)
- ☐ 자물쇠 (위탁수하물 TSA 잠금)
- ☐ 접이식 에코백/보조가방 (쇼핑용)
- ☐ 수첩/메모장 + 볼펜
- ☐ 여행용 지갑 (여권·카드 분산 보관)

## 9. 서류 & 금융

### 서류
- ☐ 여권 (유효기간 6개월 이상 확인)
- ☐ 여권 사본 x 2 (분실 대비, 별도 보관)
- ☐ 항공권 e-티켓 (인쇄 + 스마트폰 저장)
- ☐ 호주 ETA 비자 (전자비자, 사전 신청)
- ☐ 숙소 예약 확인서 (전체)
- ☐ 렌터카 예약 확인서
- ☐ 국제운전면허증 (IDP)
- ☐ 여행자 보험 증서 (영문)
- ☐ 영문 처방전 (개인 약물 소지 시)
- ☐ 긴급 연락처 목록 (대사관, 보험사, 렌터카 회사)

### 금융
- ☐ 신용카드 (해외 결제 가능, VISA/Mastercard) x 2
- ☐ 체크카드/트래블카드 (환율 우대)
- ☐ 현금 AUD 200~300 (소액, 비상용)
- ☐ 한국 원화 소액 (귀국 시 교통비)

> **Tip**: 호주는 카드 결제가 매우 보편적 (길거리 버스킹도 카드 가능). 현금은 최소한만 준비. Tap & Pay (비접촉 결제) 활성화 확인.

## 10. 짐싸기 전략 & 무게 배분

### 위탁수하물 (캐리어 2개, 각 23kg)
| 내용물 | 비고 |
|--------|------|
| 대부분의 의류 | 압축팩 활용 |
| 트레킹 폴 (접이식) | 기내 반입 불가 |
| 세면도구 (100ml 초과분) | 선크림 대용량 등 |
| 신발 (트레킹화 or 운동화) | 무거운 신발은 신고 탑승도 고려 |
| 쿨러백, 담요 | 접이식 |
| 어댑터, 멀티탭, 충전기 | |

### 기내수하물 (배낭/캐리어 + 개인가방)
| 내용물 | 비고 |
|--------|------|
| 여권, 지갑, 서류 | 개인가방에 |
| 보조배터리 (100Wh 이하) | **반드시 기내** -- 절연 보관 |
| 카메라, 노트북 등 전자기기 | 파손 방지 |
| 상비약 (기내 복용분) | |
| 기내 액체류 지퍼백 | |
| 목베개, 안대, 이어폰 | |
| 가디건/얇은 자켓 | 기내 냉방 대비 |
| 볼펜 | 입국카드 작성 |

### 무게 관리 요령
- 무거운 신발(트레킹화)은 **신고 탑승** -- 수하물 무게 절약
- 두꺼운 아우터(패딩, 고어텍스)는 **입고 탑승**
- 세면도구는 현지 구매로 무게 절약 (Chemist Warehouse, Woolworths)
- 2인 캐리어를 교차 패킹하면 한 개 분실 시에도 최소 의류 확보
