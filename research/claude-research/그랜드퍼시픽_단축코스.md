# Day 6 그랜드퍼시픽 드라이브 — ① 원본 / ② 단축 + 도로 가이드

> 6조 Day 6(5/29) 그랜드퍼시픽 구간의 두 운용안과 도로 주의사항·전환 규칙. 실행 SSOT는 [`docs/ITINERARY.md`](../../docs/ITINERARY.md) Day 6 ①/② 섹션.

---

## 1. 지리·방향 원칙

- 그랜드 퍼시픽 드라이브는 **시드니 *남쪽*** 회랑(스탠웰탑스 ↔ 울릉공 ↔ 키아마). Day 6의 "내려간다/올라온다"는 이 회랑 안의 남↔북.
- **방향 원칙**: 내려갈 때(남하) = 그랜드퍼시픽 **해안도로** / 올 때(북상·복귀) = **M1 고속도로**.
- **발드 힐 룩아웃 = 스탠웰 탑스 전망대** (동일 장소, 그랜드퍼시픽 북쪽 시작점·패러글라이딩·일몰 명소).
- **씨클리프 브릿지는 Lawrence Hargrave Drive(B65) 위에만 존재** → 내비 경유지로 넣으면 해안도로가 강제된다(검증·강제의 핵심 도구).
- 로열 내셔널 파크(RNP)는 두 버전 모두 미경유 (카툼바→Heathcote Rd 진입, "올 때 고속도로" 원칙과 충돌).

## 2. 도로 거리 (driving)

| 구간 | 거리 / 시간 |
|---|---|
| 카툼바 → 스탠웰탑스 (내륙 진입) | 135km / ~2시간 |
| 스탠웰탑스 → 키아마 (해안: Lawrence Hargrave·씨클리프·Memorial Dr) | 67.7km / ~70분 |
| 키아마 → 시드니공항 (복귀 M1) | 112km / ~1시간 45분 |
| 단축: 스탠웰탑스 → 씨클리프 | 4.9km / ~7분 |
| 단축: 씨클리프 → Bulli (해안 연장) | 12.2km / ~18분 |
| 단축: Bulli → 시드니공항 (M1) | 61.9km / ~65분 |
| 초단축: 카툼바 → 공항 (M1 직행) | 105km / ~1시간 40분 |

### turn-by-turn 도로명

- **스탠웰탑스 → 키아마 (해안)**: The Drive → **Lawrence Hargrave Drive(B65, 씨클리프 브릿지 포함)** → Thirroul(Raymond/Station St) → **Princes Hwy/Memorial Drive(B65, 절벽구간)** → 울릉공 시내(Flinders/Keira/Crown St) → **Princes Motorway(M1)** → 키아마(Terralong St)
- **키아마 → 시드니공항 (복귀)**: Terralong St → **Princes Motorway/Princes Hwy(M1)** → **Princes Hwy(A1)** → Airport Dr/Link Rd
- **카툼바 → 스탠웰탑스 (진입)**: Great Western Hwy(A32) → **Western Mwy(M4)** → **Westlink(M7, 외곽순환)** → M5 → Heathcote Rd → Princes Hwy/Mwy(A1) → Lawrence Hargrave Dr

## 3. ① 원본 코스 (시간 여유 시)

스탠웰탑스(발드힐) → Lawrence Hargrave Dr·씨클리프 브릿지 → Memorial Dr 절벽 → 울릉공 → M1 → 봄보 → 키아마(일몰) → 키아마 → M1 북상 → 시드니공항 반납. 시각표는 `ITINERARY.md` Day 6 ① 표.

**내비게이션 검증법**: 목적지를 키아마로만 찍으면 자동으로 M1 최단(76km/63분)을 안내해 그랜드퍼시픽을 통째로 건너뛴다. **반드시 경유지에 `Sea Cliff Bridge`(또는 `Bald Hill Lookout`)를 추가** → 해안 경로(67.7km/~70분)로 강제. 주행 중 내비가 씨클리프 브릿지를 안 지나면 M1에 올라탄 것 → 즉시 이탈·재경로.

- 강점: A등급 4곳 + S드라이브 풀코스, 키아마 일몰.
- 약점: 정시 출발·무정체에도 반납 ~19:10 — 21:00 마감 대비 버퍼 ~1.5~2시간(좁음).

## 4. ② 단축 코스 (시간 부족 시)

발드힐 + 씨클리프(Lawrence Hargrave Dr)만, **Bulli에서 M1 북상 복귀**. 일몰은 발드힐/씨클리프.

- 유지: S 그랜드퍼시픽 진입 + 씨클리프 + 발드힐 일몰(골든아워 #4).
- 포기: 봄보 헤드랜드, 키아마 블로우홀.
- 효과: 원본 ~180km/~2h50 주행 → 단축 79km/~1h30 + 정차 2곳 생략 = **약 1.5~2시간 회수**, 반납 ~18:30.
- 비상(초단축): 14:30 이후 더 지연 시 그랜드퍼시픽 전면 생략 → 카툼바→M1 직행 105km/~1h40. 핵심 S 하이라이트 상실 — 지연이 심각할 때만.

## 5. 전환 결정 규칙

기준점 = **카툼바 점심 후 출발 시각**(정시 13:00).

| 조건 | 선택 |
|---|---|
| 카툼바 13:30 이전 출발 + 정체 없음 | ① 원본 |
| 카툼바 13:30~14:30 출발, 또는 진입 후 정체로 버퍼 위협 | ② 단축 |
| 카툼바 14:30 이후 출발 | ② 단축 + 스탠웰탑스 정차 최소화 |
| 원본 진행 중 씨클리프 출발이 16:00 초과 | 즉시 ②로 전환 (봄보·키아마 포기) |

**우선순위**: 21:00 렌터카 반납 > 키아마 일몰 > 봄보·키아마 정차.

## 6. 도로 주의사항 (변수)

- **카툼바→스탠웰탑스**: M7 외곽순환으로 CBD는 우회하나 **금요일 오후 M4/M7/M5 정체 상습** — 출발 전 실시간 확인. 정체 심하면 Hume Mwy(M31) → Picton Rd(Wilton 분기) → Mt Ousley → Bulli Pass를 내비에서 수동 지정. 주유: 해안 진입 전 마지막 적기.
- **씨클리프 브릿지**: 다리 위 정차·주차 금지 — 북단/남단 주차 후 도보.
- **복귀 M1**: 17시 이후 야간 주행이나 왕복 다차선 안전. 톨: M1 본선 무료, 시드니 진입·공항 인근 일부 유료 가능 — 렌터카 E-Tag/Linkt 포함 여부 확인. 주유: 시드니 남부(헬렌스버그/와터폴)에서 반납 전 권장.
- **우천**: 그랜드퍼시픽은 비 와도 드라이브 가능, 전망대 사진만 짧게. 단축이 자연스러운 선택.
