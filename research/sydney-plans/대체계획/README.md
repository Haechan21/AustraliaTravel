## 핵심 발견

### 5조가 만장일치 1위인 이유

1. **동선**: The Grounds(숙소 도보 16분)로 시작 → 대회전 루프 → 되돌아가는 구간 거의 없음
2. **리듬**: 정적↔활동 파동이 자연스럽고, 앉는 시간이 고르게 분포
3. **골든아워**: Blues Point Reserve — 관광객 없는 로컬 스팟에서 3대 랜드마크 한 프레임
4. **유연성**: 예약 필수 0건, 우천 시 실내 전환 가능
5. **수면**: 21:00 귀환 → 6.5시간 수면 확보
6. **프로필**: "유명지+숨은명소 반반" 선호에 가장 정확히 부합

### 3조의 논쟁 (B' 79.6 vs C' 57.5)

- **B' 관점**: "호주의 시간축(고대→식민→현대→미래)을 하루에 관통하는 유일한 서사. Midden+Vivid 점등 동시 체험은 설계의 백미."
- **C' 관점**: "16시간 풀가동, Harry's→Midden 10분 버퍼 비현실적, CQ 5회 경유. 실행 불가능에 가까움."
- **해결책**: Harry's를 포기하면 동선 34분 절약 + Midden 17:00 유지 가능 → C' 점수 크게 개선
